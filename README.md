# العصرية للنجارة - Modern Carpentry Website

## 🎯 نبذة عن المشروع
موقع إلكتروني احترافي لشركة العصرية للنجارة - متخصصون في جميع أعمال وتشطيبات النجارة بأعلى مستوى من الجودة والاحترافية.

## 🛠️ المميزات

✅ **تصميم عصري وجميل** - واجهة احترافية تعكس جودة الخدمات
✅ **دعم اللغة العربية** - كامل الموقع باللغة العربية (RTL)
✅ **متجاوب (Responsive)** - يعمل بشكل مثالي على جميع الأجهزة
✅ **تفاعلي** - تأثيرات وحركات جميلة
✅ **سهل التعديل** - كود نظيف وسهل الصيانة
✅ **محسّن للبحث** - SEO friendly structure

## 🚀 كيفية الاستخدام

1. **تحميل الملفات**
   ```
   - index.html

   ```

2. **وضع الملفات في نفس المجلد**

3. **فتح index.html في متصفح الويب**

## 🔧 التخصيص

### تغيير الألوان
عدّل المتغيرات في `styles.css`:
```css
:root {
    --primary-color: #8B6F47;
    --secondary-color: #D4AF37;
    --dark-color: #2C2C2C;
    --light-color: #F5F5F5;
}
```

### إضافة صور حقيقية
استبدل روابط الصور placeholder:
```html
<img src="YOUR_IMAGE_URL" alt="Description">
```

### تحديث بيانات الاتصال
عدّل في قسم Contact:
```html
<p>+20 رقم الهاتف</p>
<p>your-email@example.com</p>
```
